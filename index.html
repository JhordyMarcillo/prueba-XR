¿Qué es la Realidad Aumentada (AR)?

La Realidad Aumentada es una tecnología que superpone elementos virtuales generados por computadora sobre el mundo real en tiempo real. A diferencia de la Realidad Virtual (VR) que crea un entorno completamente artificial, la AR complementa la percepción del mundo físico añadiendo información, objetos 3D o animaciones que interactúan con el entorno real del usuario.

En el contexto de este proyecto, la AR permite visualizar la mascota 3D como si estuviera físicamente presente en el entorno del usuario, pudiendo caminar alrededor de ella y observarla desde todos los ángulos.

Acerca del modelo 3D

El modelo visualizado en esta página corresponde a una mascota virtual diseñada específicamente para este proyecto de Ingeniería de Software. Fue modelado en formato GLB (GL Transmission Format Binary), que es un formato contenedor que incluye geometría, materiales, texturas y animaciones en un solo archivo optimizado para la web.

Este formato permite una carga eficiente y renderizado en tiempo real con calidad visual profesional. El modelo incluye materiales PBR (Physically Based Rendering) que simulan propiedades realistas de luz y superficie, lo que mejora la integración visual tanto en el visor 3D estándar como en los entornos de Realidad Aumentada.

El modelo está optimizado para mantener un rendimiento de 60 cuadros por segundo en dispositivos modernos, garantizando una experiencia fluida tanto en el visor estándar como en los modos de realidad aumentada.

Herramientas y tecnologías utilizadas

Model Viewer

Componente web desarrollado por Google que simplifica la integración de modelos 3D y AR en sitios web. Soporta múltiples formatos como GLB y GLTF.

WebXR API

Estándar web que permite experiencias de realidad extendida (XR) directamente en el navegador, sin plugins adicionales.

Three.js

Biblioteca JavaScript de bajo nivel para gráficos 3D que utiliza WebGL como motor de renderizado (base de Model Viewer).

AR Quick Look

Tecnología nativa de iOS que permite visualizar modelos 3D en realidad aumentada en dispositivos Apple.

Scene Viewer

Implementación de AR para dispositivos Android a través de Google Play Services for AR.

GL Transmission Format

Formato estándar (GLB/glTF) para la transmisión eficiente de modelos 3D en la web, optimizado para carga progresiva.

Controles interactivos disponibles

Rotación manual

Arrastre con un dedo (touch) o con el ratón (click y arrastre) para rotar el modelo 360 grados.

Zoom

Pellizco con dos dedos (touch) o rueda del ratón para acercar o alejar la vista del modelo.

Rotación automática

El modelo gira suavemente de forma autónoma cuando no hay interacción del usuario.

Sombra dinámica

El modelo proyecta sombra en el suelo virtual, mejorando la percepción de profundidad.

Modo AR

Botón específico (visible sobre el modelo en dispositivos compatibles) que activa la cámara y superpone el modelo en el entorno real.

Iluminación adaptable

Iluminación ambiental y direccional ajustable automáticamente según el entorno.
