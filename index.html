Controles AR

¿Qué es la Realidad Aumentada (AR)?
La Realidad Aumentada es una tecnología que superpone elementos virtuales generados por computadora sobre el mundo real en tiempo real. A diferencia de la Realidad Virtual (VR) que crea un entorno completamente artificial, la AR complementa la percepción del mundo físico añadiendo información, objetos 3D o animaciones que interactúan con el entorno real del usuario.
Modo Marcador Hiro
El modo Marcador Hiro utiliza un patrón de marcador fiducial (un cuadrado con bordes negros y un patrón interno) que la cámara reconoce para posicionar y orientar el modelo 3D en el espacio. Hiro es el marcador estándar de la librería AR.js y consiste en un patrón con forma de "H" estilizada. Al detectar este marcador impreso o mostrado en otra pantalla, el sistema superpone el modelo 3D exactamente sobre él, manteniendo su posición y rotación aunque la cámara se mueva.
Herramientas y tecnologías utilizadas
Este proyecto integra un conjunto de tecnologías web modernas para la visualización 3D y realidad aumentada:
  • Model Viewer: Componente web desarrollado por Google que simplifica la integración de modelos 3D y AR en sitios web. Soporta múltiples formatos como GLB y GLTF.
  • WebXR API: Estándar web que permite experiencias de realidad extendida (XR) directamente en el navegador, sin plugins adicionales.
  • Three.js (base de Model Viewer): Biblioteca JavaScript de bajo nivel para gráficos 3D que utiliza WebGL como motor de renderizado.
  • AR Quick Look: Tecnología nativa de iOS que permite visualizar modelos 3D en realidad aumentada en dispositivos Apple.
  • Scene Viewer: Implementación de AR para dispositivos Android a través de Google Play Services for AR.
  • GL Transmission Format (glTF/GLB): Formato estándar para la transmisión eficiente de modelos 3D en la web, optimizado para carga progresiva y renderizado en tiempo real.
Controles interactivos disponibles
  • Rotación manual: Arrastre con un dedo (touch) o con el ratón (click y arrastre) para rotar el modelo 360 grados.
  • Zoom: Pellizco con dos dedos (touch) o rueda del ratón para acercar o alejar la vista del modelo.
  • Rotación automática: El modelo gira suavemente de forma autónoma cuando no hay interacción del usuario.
  • Sombra dinámica: El modelo proyecta sombra en el suelo virtual, mejorando la percepción de profundidad.
  • Modo AR: Botón específico (visible sobre el modelo en dispositivos compatibles) que activa la cámara y superpone el modelo en el entorno real.
  • Control de iluminación: Iluminación ambiental y direccional ajustable automáticamente según el entorno.

Peso del modelo

Acerca del modelo 3D: El modelo visualizado en esta página corresponde a una mascota virtual diseñada específicamente para este proyecto de Ingeniería de Software. Fue modelado en formato GLB (GL Transmission Format Binary), que es un formato contenedor que incluye geometría, materiales, texturas y animaciones en un solo archivo optimizado para la web. Este formato permite una carga eficiente y renderizado en tiempo real con calidad visual profesional. El modelo incluye materiales PBR (Physically Based Rendering) que simulan propiedades realistas de luz y superficie, lo que mejora la integración visual tanto en el visor 3D estándar como en los entornos de Realidad Aumentada.
